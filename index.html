GriefLog

This is the GriefLog plugin for bukkit powered minecraft servers.

How to install

To install the plugin on a bukkit server just put it in the plugins directory. You can find the log file in the root directory of your server.

See the BukkitDev page for more info: http://dev.bukkit.org/server-mods/grieflog/
